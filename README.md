# Eduardo Evaristo - Portfolio

Este repositório contém o código-fonte do meu portfólio pessoal. O site foi desenvolvido para apresentar minhas habilidades, experiência profissional e projetos que desenvolvi ao longo da minha carreira como Desenvolvedor Full Stack.

## Sobre o Projeto

O portfólio é uma aplicação web estática, construída com HTML, CSS e JavaScript, que destaca minhas competências e trajetória profissional. Ele foi projetado para ser uma apresentação clara e objetiva do meu trabalho.

## Seções

O site está dividido nas seguintes seções:
*   **Home:** Uma breve introdução sobre mim e minha função.
*   **Sobre:** Detalhes sobre minha experiência, formação acadêmica e minhas paixões no mundo do desenvolvimento.
*   **Skills:** Uma lista das tecnologias e ferramentas com as quais tenho experiência.
*   **Projetos Pessoais:** Uma galeria com alguns dos projetos que desenvolvi, com links para o site em produção e para o repositório no GitHub (quando aplicável).
*   **Contato:** Informações e um formulário para entrar em contato.

## Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas na construção deste portfólio:

*   **HTML5:** Para a estruturação do conteúdo.
*   **CSS3:** Para a estilização e o design.
*   **JavaScript:** Para a interatividade e animações.
*   **Font Awesome:** Para os ícones.
*   **Google Fonts:** Para as fontes utilizadas no projeto.

## Meus Principais Conhecimentos

*   **Frontend:** Vue.js, JavaScript, HTML5, CSS3, Sass, Bootstrap.
*   **Backend:** PHP, Laravel, CodeIgniter, Golang, Python, Node.js.
*   **Bancos de Dados:** PostgreSQL, MySQL, MongoDB.
*   **Ferramentas e DevOps:** Git, GitHub, GitLab, Docker, VS Code, PHPStorm, Datagrip.

## Como Executar o Projeto Localmente

1.  Clone este repositório:
    ```bash
    git clone https://github.com/edueevaristo/portfolio.git
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd portfolio
    ```
3.  Abra o arquivo `index.html` em seu navegador de preferência.
